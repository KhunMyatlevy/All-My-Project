Myoma youth site

This website is a static blog platform I created using HTML, CSS, and JavaScript. It allows users to browse and view blog posts in a clean, responsive design. The layout ensures a user-friendly experience across various devices, making content easy to access and read. The project showcases my skills in front-end web development, particularly in structuring and styling web pages.

Key Features:

  • The website features an all-white design enhanced with shadow boxes and various 3D elements for a visually appealing look.
  • A navigation bar is included to allow easy access and seamless navigation across different pages.
  • The design emphasizes simplicity while maintaining a modern and engaging user interface.

Technologies Used:

  • JavaScript: Used to loop through posts and dynamically display content on the page.
  • HTML: Utilized for structuring the layout of the blog website.
  • CSS: Applied for styling, including the all-white design, shadow boxes, and 3D elements for a modern look.
